# Typography

Halo typography is a customizable scale of font styles for your product. There are tokens and classes that are available for assignment to any element. We do not assign styles to elements so that we can continue to create heirarchy on our pages to maintain accessibility.


Tokens are output into properties and values applied as CSS class rules.

### **Typography Tokens**

All typography tokens must now start with `typographyStyles-`.

#### Example:
- **Figma Token**: `display/medium`  
- **Design System Token**: `typographyStyles-display-medium`


## Scale

Our type scale auto generates classes based on token input.
