## Package Setup
1. Install the desired Atlas packages:
  ```bash
    npm install atlas-cdk@0.3.0
    ```
2. Use the modules in your Angular application:
    
    ```jsx
    import { CdkModule } from 'atlas-cdk';
    ```

## Theme Setup

Now that your component packages are setup and you have your `node_modules` installed from Halo's AWS, you can now setup your theme.
In Atlas, theming is a driving force of style sets for your application. Themes are the bridge between design and development. In atlas-cdk package, we have a ThemeService that you can learn more about here.

## CDK Assets

System level assets are available for use in the project and are stored in the CDK. In this directory, you will find all of the basic svg, logos, docs and images that can be used by components. Before consuming the assets, it is necessary to set up the path into the project's main build. 

Add this configuration into `angular.json`:

```
  "assets": [
    {
      "glob":"**/*",
      "input": "projects/cdk/assets/logos",
      "output": "atlas-logos"
    },
    {
      "glob":"**/*",
      "input": "projects/cdk/assets/icons",
      "output": "atlas-icons"
    },
    {
      "glob":"**/*",
      "input": "projects/cdk/assets/docs",
      "output": "atlas-docs"
    },
    {
      "glob":"**/*",
      "input": "projects/cdk/assets/images",
      "output": "atlas-images"
    }
  ]
```
