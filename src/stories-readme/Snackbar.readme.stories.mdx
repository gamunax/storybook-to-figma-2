## Usage

Import dependencies for the library into your ```app.module.ts``` file

```ts

import { SnackbarModule } from 'atlas-snackbar';

```

## API

### Snackbar

Import ```SnackbarService``` and call in your ```constructor```

```ts
constructor(
    ...
    public _snackbar: SnackbarService
    ...
  ) {...}
```
To call the service to show the ```SnackbarComponent``` must to call the following service

```ts
   this._snackbar.create(SnackbarConfig);
```
the method above creates the ```SnackbarComponent``` with the following config:

```
  SnackbarConfig {
      variant?: string;
      elevation?: string;
      message: string;
      verticalPosition?: string;
      horizontalPosition?: string;
      autoDismiss?: boolean;
      enableAction?: boolean;
      actionLabel: string;
      actionClick?: Function;
      closeClick?: Function;
  }
```
