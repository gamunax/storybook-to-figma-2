# Field

Field is a grouped package that contains [key subcomponents](#sub-components) used to capture data in forms and dialogs. Our field package offers many different sub components and capabilities.


## Usage

Import [Field dependencies](#dependencies) for the library into your ```app.module.ts``` file

```ts

import {  FieldModule } from 'atlas-field';

```
Import into `styles.scss` for a correct `select component` style's behavior

```css
  @import '@angular/cdk/overlay-prebuilt.css';
```

> https://www.npmjs.com/package/luxon

> ```bash
> npm i luxon --save
> npm i @types/luxon --save
> ```
> ```bash

## API

### Field

<ArgsTable of={ FieldComponent } />

> You can use typeahead in the field components. An enhanced text input in a panel of suggested options.
> Also in known as Autocomplete
>
> ```html
> <atlas-field>
>    <atlas-field-label>Country</atlas-field-label>
>    <atlas-select [typeahead]="true" (typeaheadChange)="filterCountry($event)" [placeholder]="'Select a country'" [(ngModel)]="country">
>      <atlas-option *ngFor="let country of countriesFiltered | async" [value]="country.value">{{ country.name }}</atlas-option>
>    </atlas-select>
>  </atlas-field>
> ```
