# Modal

Modal appear in front of the page content to displays HIGH PRIORITY information or ask for a decision. Modal disable all page functionality when they appear, and remain on screen until confirmed, dismissed, or a required action has been taken.

## Usage

Modal component require of BrowserAnimationsModule module

Import the `BrowserAnimationsModule` from `@angular/platform-browser/animations` into `app.module.ts`

```ts
import { BrowserAnimationsModule } from '@angular/platform-browser/animations';
```
Read more about: https://angular.io/guide/animations

Import into `styles.scss` for a correct `modal` style's behavior

```css
  @import '@angular/cdk/overlay-prebuilt.css';
```

Import dependencies for the library into your ```app.module.ts``` file

```ts

import { ModalModule } from 'atlas-modal';

```
