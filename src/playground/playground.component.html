Input MOSUI

Text Input with Label Animation Add the .mos-u-input--animated-input to your text input and the .mos-u-input--animated class as a wrapper around your text input and label to create an animated placeholder to label transition. *This is not supported for version 2 form field styles. Default version 2 form field styles will be applied.
Password Field - PW Redaction Directive Password redaction functinoality can be bound to any password type input by using the haloPasswordRedaction directive. Using a reference, you'll also need to link the field to the directive's haloPasswordRedaction exportAs value.
Textarea Use the textarea form element for long form text input such as a comments field. Textareas are not resizable by default but a utility class of mos-u-resize--vertical can be added to the textarea to make it resizable. If you make your textarea resizable you will have to use the paragraph style help text and not the data attribute driven help text from the label. To use hint text use the utility class as it's own element below the textarea.
