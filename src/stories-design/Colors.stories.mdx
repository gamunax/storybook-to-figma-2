# Colors
Halo color system has two distinct sets of colors that you can use to build your brand. The palette is the backbone of your color scale. The theme color tokens directly use the palette in order to create a flowing set of themes across your applications.

## Theme Tokens
The theme set is used for direct integration with our components and application styles.

## ðŸš€ **Mapping Figma Tokens to Design System Tokens**

For teams using **Figma** design tokens in version `8.0.0`, all token names in the **Atlas Design System** must now begin with `SemanticColor-` to ensure consistency and alignment with our new theme structure.

### Example

- **Figma Token**: `background/caution/strong-rest`  
- **Design System Token**: `SemanticColor-background-caution-strong-rest`
