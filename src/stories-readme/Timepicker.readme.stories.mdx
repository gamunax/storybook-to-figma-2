# Timepicker

Time pickers help users select and set a specific time.


## Usage

Field (Timepicker) component require of BrowserAnimationsModule module 

```ts
import { BrowserAnimationsModule } from '@angular/platform-browser/animations';
```
Read more about: https://angular.io/guide/animations


Field (Timepicker) component require of luxon dependency

Read more about: https://www.npmjs.com/package/luxon
