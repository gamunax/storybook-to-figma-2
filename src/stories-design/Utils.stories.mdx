# Utilities

Our utility classes provide style sets for your application that are tied to our design token language. This helps create reuse and consistency throughout your end product.

## Spacing

Spacer classes allow you to apply any `$spacing` scale token values (seen below) to `margin` and `padding` in any direction (left, right, top, bottom, x axis and y axis) on particular elements. The class name format is `.PROPERTY-DIRECTION(s)-TOKEN`

Basic:

Set the `bottom` margin to `spacing.vertical-8` value in the active theme.

```html
    <div class="margin-bottom-8"></div>
```

Multi directional:
Set the `x` axis (both left and right) padding to `spacing.horizontal-8` value in the active theme.

```html
    <div class="padding-x-8"></div>
```


## Spacing Scale

This is the scale that is used applied across the spacing class values above. These can be used across your application CSS to maintain the theme scaling and rhythm.

Base reference:
```css
--scale-content-spacing-base: 2px
```
